2. 范数(Norm)

定义：范数是任意函数且满足以下四个性质:

  1. 非负性：​
  2. 正定性： 当且仅当时成立
  3. 齐次性：​
  4. 三角不等式 ​

一些范数的例子:

One Norm $||\vv x||1 = \sum{i=1}^n|x_i|$​

Infinity Norm ​

General P Norm $||\vv x||p = \Big(\sum{i=1}^nx_i^p\Big)^{1/p}$​ ​

Frobenius Matrix Norm $||A||F = \sqrt{\sum{i=1}^m\sum_{j=1}^nA_{ij}^2} = \sqrt{tr(A^TA)}$

对于Matrix Norm来讲还有一些导出的形式:

By the Euclidean vector norm: $||A||2 = \max\limits{||\vv{x}||2=1}||A\vv{x}||2 = \sqrt{\lambda{max}}= \sigma{max}$

这个证明是trival的, 考虑
$$
||A\vv{x}||2 = \sqrt{\vv x^TA^TA\vv x} = \sqrt{\vv x^{T} (A^TA) \vv x} = \sqrt{\vv x^T P^TMP\vv x} = \sqrt{\sum{i} \lambda_i\cdot(p_ix_i)^2} \le \sqrt{\lambda_{max}} =\sigma_{max}
$$
By the vector 1-norm: $||A||1 = \max\limits{||\vv x||=1}||A\vv x||1 = \max\limits{j}\sum\limits_{i}a_{ij}=$ the largest absolute column sum.

By the vector -norm: $||A||\infty = \max\limits{||x||\infty=1}||A\vv x||{\infty} = \max_{i}\sum_{j}|a_{ij}|=$ the largest absolute row sum.

  • 的列向量是的特征向量.
  • 的非零对角元是$A^AAA^$的非零特征值的平方根(奇异值).

我们证明上述的性质:考虑
$$
A = U\Sigma V^* \Rightarrow AA^* = U\Sigma V^\cdot(V\Sigma^U^) = U(\Sigma\Sigma^)U^*

A^A = V(\Sigma^\Sigma)V^*
$$
不难证明$\Sigma^\Sigma\Sigma\Sigma^$都是对角阵, 且两者的非零元素完全相同, 其余对角线上都是0

我们处理的方法是, 首先将排序, 然后各自求得两个特征空间
$$
\begin{align*}
(A^A)v_i = \lambda_iv_i\
(AA^)u_i = \lambda_iu_i
\end{align*}
$$
如此我们会得到个和个, 做好归一化后依次填入矩阵和矩阵即可, 再求出的特征值解出对应的奇异值即可.
